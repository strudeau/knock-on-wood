Kreg Jig Drill Bit Collar Depth, A Mathematical Model

Feb 29, 2020 6 min read

I am quite new to woodworking and have had in mind of building bedroom furnitures using pocket-hole joinery. Fast, easy, reliable, I bough all three Kreg Jig® Drill Guide Blocks (standard, Micro-PocketTM, HD) that can fit in the Portable Base adapter. I though about buying a K4 Kreg Jig® but decided against because that was just one too many things to purchase for a "small project". Turns out that the different drill guide blocks don’t come with clear instructions when it comes to setting the drill bit collar depth. Good thing I had purchased a Kreg® Easy-Set Pocket-Hole Drill Bit which does contain markings on the drill bit itself as to where to set the collar depth for three wood thickness (in inches), \(\frac{1}{2}\), \(\frac{3}{4}\), \(1\frac{1}{2}\). But what about the other wood thickness, did I ask myself? Also, in all the instructions manuals I read, they were all assuming you would be joining pieces of wood of equal thickness. What about if I want to join a \(1\) inch board with a \(\frac{1}{2}\) inch one (fictional use case)? I couldn’t find anywhere clear guiding instructions on how to set the drill bit for that use case. So I ended up modelling my use case using Autodesk® Fusion 360TM to try to derive a mathematical model.

But why those specific drill bit collar depth, why not other values did I ask myself…​ It must have something to do with the screw length.

I reasonned that we would want the screw to exit out of the input wood piece at the mid point of the board’s thickness and we would want half of the screw to be in the "input" board (A) and the other half into the "destination" board (B).

According to all the documentation I could find out there and a quick check with a protractor, the angle at which the Pocket-Hole drill bit enters the wood is \(\bbox[yellow]{15^\circ}\).

Analysis

In order to maintain constant the height of the Kreg Jig® Portable Base as well as the Drill Bit Collar Depth obtained through direct measurements, I uncovered that not always does half of the screw penetrates the "destination" board such as for \(\frac{1}{2}\) and \(\frac{3}{4}\) material. In order for the drill hole to go through the center of the board, the screw must be pulled out or digged through the center by a certain offset on certain occasions. I suspect this is to accomodate cases such as for \(\frac{1}{2}\) boards where a \(\frac{3}{4}\) or \(1\) inch screw can be used.

Calculating the Drill Bit Collar Depth as a Function of the Screw Length and Input Board Material Thickness

Now on to what we were looking to find…​ Given an input board (A) and destination board (B) of different thickness, what is the Drill Bit Collar depth value I must use?

A few things to consider:

  • The Kreg Jig® Portable Base height setting is define by the input board thickness given that we want to drill through the middle of the input board.

  • The drill bit collar depth depends on how much length of the screw remains in the input board.

We can model this situation as a system of equations

Figure 4. triangle modeling
\[\begin{array}{rcl} m &=& \text{input board material thickness. e.g.:} \frac{1}{2} \\ s &=& \text{screw length e.g.:} \frac{3}{4} \\ o &\approx& \text{screw offset e.g.:} \frac{3}{16} \\ \lvert{AD}\rvert &\approx& \text{drill bit collar depth e.g.:} 3\frac{3}{16} \\ \lvert{EB}\rvert &=& (\frac{s}{2} + o)\cos(75^\circ) \\ \lvert{FD}\rvert &\approx& \lvert{AD}\rvert\sin(15^\circ) \\ \lvert{CG}\rvert &=& \text{distance between A and right edge of the block, a constant} \\ &=& \lvert{FD}\rvert - (\frac{m}{2} - \lvert{EB}\rvert) \\ &=& \lvert{AD}\rvert\sin(15^\circ) - (\frac{m}{2} - (\frac{s}{2} + o)\cos(75^\circ)) \\ \end{array}\]

Using \(\lvert{CG}\rvert\), we can express the drill bit collar depth (\(\lvert{AD}\rvert = d^\prime\)) as a function of screw length \(s\) and input board thickness \(m\).

\[\begin{array}{rcl} d &=& \frac{\lvert{CG}\rvert + (\frac{m}{2} - (\frac{s}{2} + o)\cos(75^\circ))}{\sin(15^\circ)} \end{array}\]

We notice that although \(\lvert{CG}\rvert\) should have been constant, it is not. Probably the measurement of my reference drill bit collar depth \(\lvert{AD}\rvert\) is not accurate enough. This value varies from \(\frac{21}{32}\) to \(\frac{23}{32}\), thus since this is a constant, I will be using the following value moving forward:

\[\begin{array}{rcl} \lvert{CG}\rvert &\approx& \frac{22}{32} \end{array}\]

Drill Bit Collar Depth smaller than \(3\frac{3}{16}\) are crossed out because they are not valid values because smaller values than that and your drill bit is not even going through the jig drill block at all!

The screw length corresponding to the destination board thickness were obtained from: https://www.kregtool.com/about-us/news/kreg-tool-tips-joining/choose-the-correct-screw-length.aspx
Keep in mind that the values in the table above are estimates and you should treat them as such. They are obtained by calculation, measurements and contain a fair bit of uncertainties and guesswork. They DO NOT represent your jig’s manufacturer recommendations. If you are building anything in a professional capacity or that must comply with building regulations, for example, DO NOT use those values and contact your jig’s manufacturer for the exact information.
